SPRINT 2D - MOTORE COMMERCIALE MEXAL

1. Estrarre lo ZIP nella cartella principale del progetto.
2. Confermare la sostituzione dei file esistenti.
3. In Supabase SQL Editor eseguire:
   supabase/migrations/20260719090000_sprint_2d_commercial_engine.sql
4. Fermare Vite con Ctrl+C.
5. Eseguire npm run dev.
6. Aggiornare il browser con Ctrl+F5.

File sostituiti:
- src/modules/orders/pages/NewOrder.jsx
- src/modules/orders/services/priceEngine.js
- src/modules/orders/orders-module.css

Funzioni:
- listino cliente
- particolarità prezzo
- particolarità sconto
- matrice sconti
- sconto pagamento
- dettaglio calcolo espandibile per ogni riga
- nessun margine e nessun ricarico

Verifica tecnica eseguita:
- npm run build completato correttamente con Vite 8.1.3.
